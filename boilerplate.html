IO.inspect/2

basic console out


Example:

iex(1)> map = %{key0: :value0, key1: :value1}

%{key0: :value0, key1: :value1}

iex(2)> IO.inspect(map)

%{key0: :value0, key1: :value1} ← output to console

%{key0: :value0, key1: :value1} ← returns the thing inspected
